#### 1)RDBMS -- RELATIONAL DATABASE MANAGEMENT SYSTEM..(MYSQL, POSTGRES, ORACLE, SQ LITE).IT DEALS WITH STRUCTURED DATA.



\-->RDBMS (RELATIONAL DATABASE MANAGEMENT SYSTEM) IS A TYPE OF DATABASE SYSTEM USED TO STORE, MANAGE, AND RETRIEVE DATA IN THE FORM OF TABLES (RELATIONS).



\-ORGANIZES DATA INTO ROWS AND COLUMNS (TABLES)



\-MAINTAINS RELATIONSHIPS BETWEEN DIFFERENT TABLES.



\-USES SQL (STRUCTURED QUERY LANGUAGE) TO INTERACT WITH DATA.
